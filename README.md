# beat-js-slider

## intro
- javascript library for slider
- package

## settings
```
interface Option {
	classNames?: {
		init?: string
		image?: string
	},
	scroll?: {
		interval?: number | default 3000
		direction?: string | default left
    use?: boolean | default true
	},
}
```

## run command
```
./run.sh
```

## webserver
```
open http://localhost:3000
```
